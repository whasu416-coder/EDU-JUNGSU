[목표] 교육과정정보 홈페이지 제작

[연관정보] sample_data/course_info.csv의 모든 정보를 기반으로 제작. 각 행의 section, subsection, label, content, detail, meta, link_url 값을 읽어 홈페이지에 맞게 구조화할 것

[홈페이지 분위기] apple.com과 비슷한 느낌의 미니멀하고 세련된 분위기. 넓은 여백, 큰 타이포그래피, 깔끔한 정보 배치, 고급스럽고 차분한 색감 사용. 과한 장식, 복잡한 패턴, 무거운 그림자, 불필요한 애니메이션은 지양

[디자인 방향] 단순한 회사 공지 페이지가 아니라, 교육 안내와 신청자 정보 확인에 최적화된 프리미엄 단일 페이지 웹사이트로 제작. 첫 화면에서 과정명, 일정, 장소, 핵심 메시지가 명확히 보여야 함

[필수 섹션] 히어로 영역, 과정 개요, 교육 대상, 핵심 메시지, 시간표, 세션별 안내, 준비사항 및 사전안내, 교육장소, 인근식당, FAQ, 문의처, 자료 다운로드, QR 또는 외부링크 안내

[콘텐츠 구성] course_info.csv의 display_order를 기준으로 정렬하고, 같은 section끼리 자연스럽게 묶어서 보여줄 것. 단순 CSV 나열처럼 보이지 않게, 사용자가 읽기 쉬운 카드형 또는 섹션형 UI로 재구성할 것

[기능] 전화번호 클릭 시 전화 연결(tel:), 이메일 클릭 시 메일 작성(mailto:), 장소 버튼 클릭 시 약도보기 링크로 이동, 다운로드 링크 클릭 가능, 외부 링크는 새 창으로 열기

[교육장소 처리] 교육장소는 버튼형 CTA로 강조하고, 버튼 문구는 “약도 보기” 또는 “교육장소 확인하기”처럼 명확하게 표시할 것

[인근식당 처리] 식당명, 메뉴 및 업종, 건물 내 위치, 상세 정보 링크를 표 또는 카드 목록으로 정리. 점심시간에 빠르게 확인할 수 있도록 가독성을 높일 것

[모바일 대응] 모바일 우선으로 설계하되 데스크탑에서도 고급스럽게 보이도록 반응형으로 제작. 모바일에서는 주요 정보가 첫 화면 안에 최대한 들어오도록 구성

[문장 톤앤매너] 안내문은 친절하고 명확하게 작성하되, 너무 딱딱하지 않게. 교육 운영용 안내 페이지답게 신뢰감 있고 정돈된 표현 사용. 불필요하게 마케팅 문구가 많지 않도록 할 것

[출력물 조건] 실제 배포 가능한 수준의 HTML 기반 결과물로 작성. 시맨틱한 구조를 사용하고, 섹션명/버튼명/안내문구는 모두 한국어로 작성할 것

[중요] 정보 누락 없이 course_info.csv의 내용을 최대한 반영하되, 사용자 경험 관점에서 우선순위를 재배치해도 됨
